## Run a demo 

1. Clone parrot repo:
  
  ```bash
  git clone git@github.com:xc0d3rz/Parrot.git && cd Parrot
  ```

2. Install dependencies
  
  ```bash
  npm install 
  ```
3. Start the webserver

  ```bash
  npm start
  ```
4. Go to the webserver
  http://localhost:3000
